How to Use WireGuard

Import your .conf file and connect in under 2 minutes.

Step 1 — Subscribe & Receive Your Config File

After completing your order on the Pricing page, you will receive an email containing:

  • A .conf configuration file attachment
  • A QR code image (for mobile devices)
  • Your chosen server location and plan details

The .conf file looks like this:

[Interface]
PrivateKey = <your-private-key>
Address = 10.0.0.2/32
DNS = 1.1.1.1

[Peer]
PublicKey = <server-public-key>
Endpoint = <server-ip>:51820
AllowedIPs = 0.0.0.0/0, ::/0
PersistentKeepalive = 25

Step 2 — Install WireGuard & Import Config

1
Download and install WireGuard from wireguard.com.
2
Open WireGuard. Click "Add Tunnel" → "Import tunnel(s) from file".
3
Browse to the .conf file you received by email and select it.
4
The tunnel appears in the list. Click "Activate" to connect.
5
The status changes to Active. You are now connected and protected.
1
Install WireGuard from the Mac App Store.
2
Open WireGuard. Click the "+" button → "Import tunnel(s) from file".
3
Select the .conf file from your Downloads folder.
4
Allow the VPN configuration when macOS prompts you.
5
Toggle the tunnel ON. You're connected.
1
Install WireGuard from the App Store.
2
Open the email on your iPhone. Tap the .conf attachment or open the QR code image.
3
Option A (QR Code): In WireGuard tap "+" → "Create from QR code" and scan the QR code.
4
Option B (File): Tap the .conf file → Share → WireGuard to import directly.
5
Allow the VPN configuration. Toggle the tunnel ON.
1
Install WireGuard from Google Play.
2
Open WireGuard. Tap the "+" button.
3
Option A (QR Code): Tap "Scan from QR code" and scan the QR code from your email.
4
Option B (File): Tap "Import from file or archive" and select the .conf file.
5
Allow the VPN permission. Toggle the tunnel ON.
1
Install WireGuard via your package manager:
# Ubuntu / Debian
sudo apt install wireguard

# Fedora
sudo dnf install wireguard-tools

# Arch Linux
sudo pacman -S wireguard-tools
2
Copy the .conf file to /etc/wireguard/:
sudo cp ~/Downloads/wg0.conf /etc/wireguard/wg0.conf
sudo chmod 600 /etc/wireguard/wg0.conf
3
Start the tunnel:
sudo wg-quick up wg0
4
To auto-start on boot:
sudo systemctl enable wg-quick@wg0
5
To disconnect: sudo wg-quick down wg0
